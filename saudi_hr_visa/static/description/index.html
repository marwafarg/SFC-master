Employee Visa Request

Manage all employee's visa request for any business or personal trip.
Behalf of employee, HR department will manage visa process with Government/Visa council.

Visa Request

For individual or family, employee can put visa request.
There are different type of visa request:
  1. Exit re-entry Visa
  2. Final Exit
  3. Extension of Exit re-entry Visa

Based on visa request type system will ask purpose of visa, old visa number, client name and so on information.


Validation

HR officer will check and validate employee visa request.



Apply in process

After validation HR Officer can put as in process and applied for visa.



Issued Visa


HR Officer will update the status which received from visa council.Employee will collect the visa from Officer once status updated.


Generate Expenses


If any type of expense occur then expense manager will add expense (company & employee contribution) details and generate the expense.
